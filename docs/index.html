What is an AI Agent?

An autonomous entity that uses Large Language Models (LLMs) to reason, use tools, and complete complex tasks to achieve a specific goal.

Types of AI Agents

According to IBM, agents range from simple reflex systems to complex learning systems.

  • Simple Reflex Agents
  • Model-based Reflex Agents
  • Goal-based Agents
  • Utility-based Agents
  • Learning Agents

crewAI primarily focuses on Goal-based and Utility-based collaboration.

Source: IBM Think Topics

Single vs. Multi-Agent

Single LLM

Asking one person to do everything (Research, Analysis, Writing, Proofreading).

Multi-Agent (crewAI)

Assigning tasks to specialized agents who collaborate like a professional team.

Analogy: A professional Newsroom. You have a Reporter, an Editor, and a Fact-checker working together, rather than a single blogger trying to do it all.

How crewAI Works

Built on top of LangChain with a modular design principle.

Core Components:

  • Agents: The specialized workers.
  • Tasks: The assignments.
  • Tools: The equipment (Web search, APIs).
  • Process: The workflow (Sequential/Hierarchical).

Essentially, you define the "Who," "What," and "How," then the Crew manages the execution.

The Agent

"Who is doing the work?" — Think of it as a Job Description.

researcher:
  role: "{topic} Senior Data Researcher"
  goal: "Uncover cutting-edge developments in {topic}"
  backstory: "You're a seasoned researcher with a knack..."

The Task

"What needs to be done?" — Must be assigned to one agent.

research_task:
  description: "Conduct a thorough research about {topic}..."
  expected_output: "A list with 10 bullet points."
  agent: researcher

The Tool

"What resources can they use?"

  • Capabilities: Web Search, File Read/Write, API calls.
  • Action: Without tools, agents can only "think" (reason). With tools, they can "act."
  • Customization: Use built-in tools or create your own custom Python tools.

The Crew

"How does the team operate?"

  • Orchestration: Coordinates agents and tasks.
  • Processes:
    - Sequential: Step-by-step (A → B → C)
    - Hierarchical: A Manager agent delegates work.
  • Kickoff: Start the process with inputs to get the final result.
